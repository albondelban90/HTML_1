Definición:

Hyper Text Markup Language:

Lenguaje de programación de etiquetas, que sirve como una estructura de arbol para el contenido y desing de una pagina o aplicacion web.

Estructura HTML

Sistemas de Etiquetas dentado, que se compone por bloques.

La Etiqueta:

Consta de 3 partes:

1.Apertura: Costa de 2 simbolos de mayor < > y menor y dentro el nombre de la etiqueta

2 Contenido Entre la Apertura y el Cierre va el cotenido de la etiqueta, que puede ser: texto,imagen,vinculo..etc.

3.Cierre Es como la Apertura, solo que en el simbolo lleva
un slash diagonal como cierre < / >

Tipos de Etiquetas y sus propiedades

Existe muchas etiquetas que veremos a continuación con sus propiedades, que sirven para darle funcionalidades, estructura, diseno a las paginas web.

< DOCTYPE html>
Esta etiqueta es la primera de todo codigo html y significa el modo html-5 sin no esta mencionada en el código, el buscador entraría en modo Quirks mode, que es el modo de los 90`s (antiguo) de las paginas web.

< head >
Representa la colección del metada que no necesariamente está visible al usuario

< title >
Representa el titulos de la pagina web, es importante resaltar que los motores de busqueda utilizan el titulo para posicionar paginas web.

< body >
"Representa todo el contenido de la web"

< h1-h6 >
Heading o cabezeras y se utiliza para indicar cabeceras, subcabeceras o sub-sub-cabeceras.. de la pagina, y a pesar que van sustancialmente dreciendo su tamano, no se usa para agrandar o achicar o modificar stylo del texto sino para fragmentar el contenido de la pagina web y son utilizados por los motores de búsquedas. y la etiqueta < h7 > no existe ya que se entiende que si tu codigo tiene mas de 6 cabezeras, se considera no es viable a la semantica de un contenido.

< p >
Párrafos: aquí dentro de esta etiqueta es donde podemos colocar los textos

< br >
Esta etiqueta sirve para hacer Saltos de Linea en el texto. es decir es la accion cuando precionas Enter en tu documento de Word. y mientras mas colocas esta etique mas saltos de linea hara.

< hr >
Secciona la pagina web haciendo una division en la pagina como salto de pagina.

< !-- -- >
Dentro de esta etiqueta se deja un mensaje de comentario, que no se muestra al cliente solo se lee desde el codigo

< span >
Selección silenciosa o de referencia y se usa para identificar una parte del texto dentro de un parrafo, para asignar propiedades como estilos (CCS) y manipular la palabra es como tomar una palabra que deseas aplicarle alguna modificacion o resaltar la palabra que estaria dentro de esta etiqueta.

Esta etiqueta aparte de su funcion mencionada, se combina dentro de ella antes de cerrarla para crear propiedades como esta a continuacion: style="color: blue;"> eso le indica que la seleccion del texto dentro de la etiqueta span, se pintaria de color azul.

Formularios < form />
Ya hemos visto varias etiquetas que nos permiten mostrar informacion al usuario. Ahora vamos ver una forma de interactuar con el usuario y mas que todo una forma de Recopilar informacion con una etiqueta que se llama: < form /> donde adentro tenemos varias propiedades como < input /> que es la propiedad para generar una casilla, donde a su vez podemos incorporar otras etiquetas como proiedad para darle referencia a la casilla, y se llama < label /> que indique que tipo datos vamos a solicitar en la casilla. ejemplo el dato "Nombre" y una vez declarado el tipo de dato, con la propiedad < label, dentro vamos a colocar la propiedad for="Nombre" , mas la propiedad id= "y entre comillas colocar el mismo texto que le dimos la propiedad" a < for="" para cuando haga clic al "nombre" se coloque el cursor en la casilla. ejemplo



Dentro de la etiqueta < input vamos agregar una propiedad que se llama name="nombre", que le indicara al servidor una llave y un valor, donde la llave es name y el valor contenida es nombre, tambien otra propiedad similar a la de < label , es placeholder que colocara dentro de la casilla, el valor como sugerencia que hemos asignado.
